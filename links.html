Important Links

  • ALMA Science Portal: Explore proposals, data, tools, and documentation provided by the Atacama Large Millimeter/submillimeter Array.
  • ALMA Science Archive: Access the archival data of the Atacama Large Millimeter/submillimeter Array.
  • JWST: Explore the archival data of the James Webb Space Telescope.
  • NRAO Archive: Access archival data from the Very Large Array (VLA) and Green Bank Telescope (GBT).
  • SMA Archive: Access archival data from the Submillimeter Array (SMA).
  • GMRT Archive: Access archival data from the Giant Metrewave Radio Telescope (GMRT).
  • CASSIS: CASSIS is a free, interactive spectral analysis tool that supports LTE modelling.
  • MADCUBA: Designed for LTE spectral modeling and fitting.
  • UCLCHEM: Gas-grain chemical code for astrochemical modelling.
  • NAUTILUS: Gas-grain chemical code for astrochemical modelling.
  • Splatalogue: Database for astronomical spectroscopy.
  • KIDA: Access a database of gas-grain chemical reactions with rates.
  • UMIST: Access a database of gas-grain chemical reactions with rates.
